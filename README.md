Soy Ingeniero Civil Electrónico con experiencia en **IoT, desarrollo de hardware y software a la medida**, especialmente orientado a sistemas de monitoreo remoto y telemetría para gestión hídrica y proyectos de agricultura de precisión.

Mi foco es construir soluciones **robustas, escalables y bien documentadas**, que realmente resuelvan problemas en terreno.

## Proyectos

### API de Gestión de Productos (Bootcamp Módulos 7–9)

API REST desarrollada con **Node.js y Express**, conectada a una base de datos mediante **Sequelize**, que permite gestionar productos (crear, listar, actualizar y eliminar).

- Endpoints protegidos con validaciones y manejo adecuado de códigos HTTP.
- Estructura de proyecto organizada en capas (rutas, controladores, modelos, middlewares).
- Probada con herramientas como Postman/Insomnia.

**Tecnologías:** Node.js, Express, Sequelize, PostgreSQL/MySQL.

### Plataforma IoT para Monitoreo Hídrico (Proyecto Personal / Profesional)

Sistema de telemetría para monitorear variables como caudal, nivel y calidad de agua en zonas rurales:

- Uso de **sensores conectados a microcontroladores** (ESP32 / ATmega2560).
- Comunicación mediante **LoRa / 4G (SIM7600)** hacia un broker **MQTT (EMQX)**.
- Almacenamiento de datos en la nube y visualización a través de una aplicación web.

**Tecnologías:** ESP32/ATmega2560, LoRa, SIM7600, EMQX, AWS, Node.js, bases de datos SQL/NoSQL.

[Ir al proyecto](https://github.com/TU_USUARIO/TU_REPO_IOT) <!-- Reemplaza con tu URL real -->
